# JCSS Corporate Advisory Website

## Features Implemented

### Layout Structure
- **Logo (Top-Left)**: JCSS branding with "Delivering Transparency" tagline
- **Navigation (Center-Top)**: Home, Services, Team, Career, Case Studies, Testimonials, Contact Us
- **Language Selector (Top-Right)**: EN, HI, JA, ID with active state highlighting
- **Vertical Service Menu (Right Side)**: 
  - Corporate Advisory
  - Audit & Assurance
  - Enterprise Support
  - Tax
  - Legal
  - Contact Us

### Animations
- Fade-in effects for logo and language selector
- Slide-in animations for navigation items with staggered delays
- Slide-in-right animation for vertical service menu
- Bounce-in effect for Corporate Advisory button
- Hover effects on all interactive elements
- Smooth scrolling behavior
- Custom orange scrollbar matching brand colors

### Interactive Elements
- Active navigation highlighting with animated underline
- Language selector with active state
- Hover effects on service tabs with scale and color transitions
- Floating action button (chat icon) with pulse animation
- Responsive header that changes on scroll

### Styling
- Dark gradient background (slate-900 to slate-800)
- Orange accent color (#f97316) matching JCSS branding
- Professional typography and spacing
- Background image overlay with low opacity
- Glassmorphism effects on header when scrolled

### Content
- Corporate Advisory hero section
- Service tags (Business Modelling, Due Diligence, Structuring)
- Company description and value proposition
- Contact information (phone and email)
- Call-to-action for scheduling meetings

## To Run
```bash
npm run dev
```

Visit http://localhost:3000 to see the animated single-page site.
